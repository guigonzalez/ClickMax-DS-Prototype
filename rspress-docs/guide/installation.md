# Installation

Guide for installing and setting up the ClickMax Design System.

## Prerequisites

- Node.js 18+ or 20+
- npm 9+ or pnpm 8+

## Clone the Repository

```bash
git clone https://github.com/guigonzalez/ClickMax-DS-Prototype.git
cd ClickMax-DS-Prototype
```

## Install Dependencies

```bash
npm install
```

## Development Setup

Start one of the development servers:

```bash
# Storybook (recommended for component development)
npm run storybook

# Documentation site
npm run docs:dev

# Main playground app
npm run dev
```

## Build for Production

```bash
# Build Storybook
npm run build-storybook

# Build documentation
npm run docs:build

# Build main app
npm run build
```

For more details, see the [Getting Started](/docs/guide/getting-started) guide.
